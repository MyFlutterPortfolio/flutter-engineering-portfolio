# Flutter Engineering Portfolio

Production-oriented Flutter engineering portfolio focused on architecture, performance, security, networking, testing, design systems, reusable packages, and real application case studies.

## Featured project

### Dealio - AI shopping assistant

Dealio is a full-stack Flutter application integrated with a TypeScript/Express backend, Prisma/PostgreSQL, Redis, authentication, progressive search, AI analysis, alerts, barcode scanning, image recognition, subscriptions, and localization.

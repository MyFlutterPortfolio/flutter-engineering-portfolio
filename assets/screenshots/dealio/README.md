# Dealio screenshots

This is the correct location for Dealio screenshots in the portfolio repository.
Add the image files directly into this folder; the case study links to this
directory from `docs/projects/dealio_case_study.md`.

Recommended sequence and filenames:

1. `01-login.png` — login
2. `02-search.png` — search form
3. `03-progressive-results.png` — progressive search results
4. `04-product-detail.png` — product detail and AI analysis
5. `05-store-comparison.png` — store comparison
6. `06-profile-alerts.png` — alerts and profile

Use consistent device framing and crop each image to the relevant UI. Avoid personal data, tokens, and real customer information.
